QUESTION 2.(ii)

Bottom-k sketch

Lo sketch conserva i k elementi con ranking hash piu' piccolo.

SIZE|A| ~= (k - 1) / kth(A)
JACCARDfrazione comune nello sketch dell'unione
  • Universo: stringhe.
  • Ranking implicito: hash in [0, 1).
  • Sketch confrontabili: stesso k e stesso seed.
  • Uso dichiarato di ChatGPT come supporto per implementazione e spiegazione.
def update(self, element):
    element = str(element)
    if element in self._ranks:
        return

    rank = self._rank(element)
    if len(self._ranks) < self.k:
        self._ranks[element] = rank
        return

    worst = max(self._ranks, key=self._ranks.get)
    if rank < self._ranks[worst]:
        del self._ranks[worst]
        self._ranks[element] = rank
Spiegare l'intuizione: campione deterministico dei rank minimi, riusabile per cardinalita' e similarita'.
